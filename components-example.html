Header Example

1. Single-Line Text Field Component

Used for chapter titles, headlines, and keywords

const textField = ComponentLibrary.createSingleLineTextField({ label: 'Chapter Title:', placeholder: 'Enter chapter title...', id: 'chapterTitle', dataAttributes: { 'chapter-id': '123' } });

2. Multi-Line Text Field Component

Used for chapter outlines with auto-resize functionality

const outlineField = ComponentLibrary.createMultiLineTextField({ label: 'Chapter Outline:', placeholder: 'Enter detailed chapter outline...', id: 'chapterOutline', dataAttributes: { 'chapter-id': '123' } });

3. Rating Input Field Component

Centered rating input for scoring content (1-100)

const ratingField = ComponentLibrary.createRatingField({ label: 'Rating / 100:', id: 'contentRating', dataAttributes: { 'field': 'content_rating' } });

4. Form Row Layout Component

Combines text field and rating field in a horizontal layout

const textField = ComponentLibrary.createSingleLineTextField({...}); const ratingField = ComponentLibrary.createRatingField({...}); const formRow = ComponentLibrary.createFormRow({ textField, ratingField });

5. Gradient Button Component

Chapter number buttons with gradient styling

const chapterButton = ComponentLibrary.createGradientButton({ text: 'CHAPTER 1', onClick: () => console.log('Chapter 1 clicked'), id: 'chapter1Btn' });

6. Section Container Component

Consistent section background and spacing

const section = ComponentLibrary.createSectionContainer([ gradientButton, formRow.container ]);
